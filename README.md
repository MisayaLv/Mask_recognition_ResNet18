# 基于 PyTorch 的口罩佩戴状态识别

## 一、项目简介
本项目基于 PyTorch 实现了一个图像分类模型，用于识别图像中不同的口罩佩戴状态。
模型采用迁移学习方法，在 ImageNet 预训练模型的基础上进行微调，以提升在小规模数据集上的泛化性能。

## 二、数据集说明
- 数据集来源：Kaggle
- 数据类型：图像分类数据
- 数据划分：训练集（train）与验证集（val）
- 注：由于数据体量较大，数据未上传至 GitHub

## 三、模型与方法
- 网络结构：ResNet（ImageNet 预训练）
- 训练策略：
  - 冻结骨干网络参数
  - 微调高层特征与分类层
- 损失函数：CrossEntropyLoss
- 优化器：Adam

## 四、实验结果
- 验证集准确率约为 97%
- 模型训练过程稳定，未出现明显过拟合现象

## 五、错误分析
在验证集上统计预测结果与真实标签的差异，对错误样本数量及类别分布进行了分析，
为后续模型改进提供依据。

## 六、运行环境
